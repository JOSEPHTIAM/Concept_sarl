|||||||||||||||     LES COMMANDES   ||||||||||||||||||||||||||||||||||||||||||||||||||||||||||||


1__ Pour créer un nouveau projet :
npx create-next-app@latest nomProjet --example "https://github.com/vercel/next-learn/tree/main/dashboard/starter-example" --use-pnpm

2__ Pour pour installer les paquets du projet :
pnpm i

3__ Pour lancer le serveur :
pnpm dev

4__ Pour installez toutes les dépendances dans un projet déjà existant :
npm install
npm install @neondatabase/serverless

5__ Pour le serveur de développement local :
localhost:3000/seed
